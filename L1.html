  1.4 การเชื่อมต่ออุปกรณ์ต่าง ๆ
    เครื่องคอมพิวเตอร์สมัยใหม่มีการจัดวางตำแหน่งของพอร์ตต่าง ๆ คล้ายกับแบบเก่าในอดีต แต่พอร์ตบางชนิดอาจจะไม่ได้ใช้งานแล้ว
เช่น พอร์ตอนุกรม พอร์ตขนาน พอร์ตเกม บางเครื่องก็ลดหรือตัดพอร์ตพีเอสทู (PS/2) ออก และหันมาใช้งานพอร์ตแบบยูเอสบี (USB) แทน


  1.5 ข้อควรระวังในการใช้เครื่องคอมพิวเตอร์
 1.5.1 ไม่ควรปิดเปิดเครื่องคอมพิวเตอร์ติดต่อกันทันที
 1.5.2 ขณะใช้คอมพิวเตอร์ ไม่ควรนำอาหารและเครื่องดื่มเข้ามารับประทาน
 1.5.3 ห้ามเคาะ เขย่า เหยียบ กระโดด บนโต๊ะเครื่องคอมพิวเตอร์
 1.5.4 อย่าขีดเขียนสิ่งใดลงบนหน้าจอภาพ คีย์บอร์ด หรือส่วนใดส่วนหนึ่งของเครื่องคอมพิวเตอร์
 1.5.5 เมื่อใช้คีย์บอร์ดต้องใช้อย่างเบามือ ไม่ควรกระแทกแรง ๆ
 1.5.6 หลักจากใช้งานเสร็จ ควรคลุมเครื่องคอมพิวเตอร์ด้วยผ้าหรือพลาสติกป้องกันฝุ่น
 1.5.7 อย่าให้อากศภายในห้องร้อนและชื้นมาก

  1.6 การแก้อาการเครื่องค้าง
    การทำงานกับคอมพิวเตอร์อาจพบปัญหาโปรแกรมที่กำลังใช้งานอยู่เกิด "อาการค้าง" ไม่สามารถใช้งานได้อาการแบบนี้
เรียกว่า "ระบบแฮงก์"
  วิธีแก้ไขอาการเครื่องแฮงก์สามารถทำได้ ดังนี้
1 ) กดปุ่ม Ctrl + Alt + Del บนแป้นพิมพ์
2 ) ปรากฏหน้าต่างให้เลือกตัวเลือก Task Manager
3 ) เลือกโปรแกรมที่ค้าง (Not responding) แล้วกดปุ่ม End task
(ถ้ายังแก้ไขไม่ได้ ต้องกดปุ่ม Reset เครื่องใหม่ หรือกดปุ่ม Power ที่เครื่องค้างไว้จนไฟดับ แล้วเปิดเครื่องใหม่)

   1.7 การปิดเครื่องอย่างถูกวิธี
   การปิดเครื่องให้ถูกวิธีเป็นเรื่องที่ไม่ควรละเลย เพราะขณะทำงาน ข้อมูลบางส่วนจะถูกโหลดไว้ในหน่วยความจำแรม
RAM เมื่อปิดเครื่อง ข้อมูลในหน่วยความจำแรม RAM จะหายไปทันทีโดยที่ไม่มีทางกู้คืนมาได้ ฉะนั้นก่อนปิดเครื่องจะต้องตรวจสอบเสมอว่าบันทึกไฟล์งานที่เปิดไว้เรียบร้อยหรือไม่
   วิธีป้องกันการสูญหายของข้อมูลอย่างง่าย ๆ คือ
1 ) ทุกครั้งที่จะปิดเครื่อง ให้ออกจากโปรแกรมที่ทำงานอยู่ทั้งหมด แล้วจึงกดปุ่ม Start
2 ) ปิดเครื่องที่ปุ่ม Start > Power
3 ) คลิกที่ Shut down
