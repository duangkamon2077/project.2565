   5.การทำงานกับไดรฟ์ ไฟล์และโฟลเดอร์
  การจัดเก็บข้อมูลในเครื่องคอมพิวเตอร์
   เครื่องคอมพิวเตอร์จะทำหน้าที่จัดเก็บเอกสารข้อมูลต่าง ๆ เป็นจำนวนมาก ซึ่งเปรียบได้กับตู้เก็บเอกสารขนาดใหญ่ที่มีลิ้นซักอยู่หลาย ๆ ชั้น เพื่อใช้เก็บเอกสารให้เป็นระเบียบ
ไฟล์ (File) คือ เอกสารที่ใช้เก็บข้อมูลเรื่องต่าง ๆ เอาไว้ในเครื่องคอมพิวเตอร์
โฟลเดอร์ (Folder) คือ แฟ้มที่ใช้เก็บไฟล์เพื่อความสะดวกในการจัดหมวดหมู่
ไดรฟ์ (Drive) คือ พื้นที่เก็บเอกสารต่าง ๆ ไม่ว่าจะเป็นไฟล์ข้อมูลหรือโฟลเดอร์


  การจัดการไฟล์และโฟลเดอร์ด้วย This PC
   This PC เป็นเครื่องมือหนึ่งของตัวจัดการไฟล์ (File Explorer) ที่ใช้ในการเข้าไปค้นหาส่งต่าง ๆ ที่อยู่ในเครื่องคอมพิวเตอร์ การเรียกใช้ This PC สามารถทำได้หลายวิธี ดังนี้
วิธีที่ 1 ดับเบิลคลิกที่ไอคอน This PC บนพื้นเดสก์ท็อป
วิธีที่ 2 ใช้ปุ่ม Start คลิกที่ปุ่ม Start-เลือนลงมาที่ Windows System แล้วกดขยายออกมา-คลิกเมาส์ที่ This PC
วิธีที่ 3 ใช้ปุ่ม Search Windows คลิกที่ปุ่มSearch Windows-พิมพ์ This PC แล้วกดปุ่ม Enter
